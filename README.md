# Customer_Churn_Prediction
Customer Churn Prediction using Logistic Regression

A Machine Learning web application that predicts whether a customer will churn or stay, based on their demographic and service usage details.
This project includes data preprocessing, model training, evaluation, and a Streamlit web app for real-time prediction.

Logistic Regression model trained on Telco Churn dataset

 Cleaned & preprocessed data (handling missing values, encoding, scaling)

 Model saved using joblib

 Interactive Streamlit UI

 User inputs converted into model-ready format
